Cara Setting Wifi pada Router atau Modem ZTE

Silahkan perhatikan langkah-langkah berikut.

1. Pastikan Hp anda terhubung ke jaringan wifi

2. Buka Alamat 192.168.1.1 pada Browser Google Chrome atau Klik disini

3. Masukkan :
* Account = user
* Password = user

4. Pilih Menu Network ( perhatikan lingkaran kuning)

5. SSID name (untuk mengganti nama wi-fi)
6. WPA-Prehsared Key(untuk mengganti sandi wi-fi)
7. Lalu Klik Submit

Selesai.
apabila ada yang perlu ditanyakan Silahkan Klik tombol Chat diatas.
